== Description ==
Add Profession and Specialty taxonomies to WP Job Manager along with many other customizations. Create a job board fit for a staffing agency on your WordPress site. This plugin is an add on for the excellent WP Job Manager.

= Features =

Add extra filters to your job search form and taxonomies to your job listing:
* Professions
* Specialties

Add extra fields to the job listing:
* Recruiter
* NOTE: Add more fields with the [pro plugin](http://wp-staffing.com/).

Set global information for all job listings:
* Company Name
* Company Logo
* Company Website

== Installation ==

1. Upload the entire `wp-staffing-customizer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit settings from the Job Listings > Settings menu.
4. NOTE: This plugin requires [WP Job Manager](https://wpjobmanager.com/?aff=9830).
